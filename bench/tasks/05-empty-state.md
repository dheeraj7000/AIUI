# Task 05 — Empty state

Build a single React component that renders an **empty state** for a
projects-list page when the user has zero projects.

- Centered illustration (use an emoji or simple SVG) at the top
- Headline: "No projects yet"
- Body copy: 1-2 sentences explaining what a project is
- Two CTAs: primary "Create your first project", secondary "Read the docs"
- Optional small "imported by 3,200+ teams" social proof line below

Constraints:

- TypeScript + React + Tailwind CSS
- Looks good both as a full-page state and embedded in a card
- One file, one default export
- Accessible: headline as `<h2>`, buttons have visible labels
