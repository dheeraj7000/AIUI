# Task 01 — Pricing card

Build a single React component that renders a **3-tier pricing card layout**
(Starter, Pro, Enterprise) for a SaaS product.

Constraints:

- TypeScript + React + Tailwind CSS
- Each tier shows: name, monthly price, 4–5 feature bullets, a CTA button
- The middle tier (Pro) should be visually emphasised (border, badge, or
  background)
- Should be responsive — stacks on mobile, three columns on desktop
- One file, one default export
- No external UI library imports; use Tailwind utilities only

Use sensible content for the bullets — placeholder is fine.
