# Task 07 — Marketing hero

Build a **marketing hero section** for a SaaS landing page.

- Eyebrow tag (e.g. "NEW · v2 just shipped")
- Big headline (~3-4 lines, ~10-15 words)
- Subhead paragraph
- Two CTAs: primary "Start free" + secondary "Watch demo" (with play icon)
- Right side: a dashboard mockup / screenshot placeholder (use a simple
  bordered box with a few colored rectangles representing UI)
- Background: subtle gradient or pattern, NOT a solid white
- Logos row underneath: "Trusted by" + 5 placeholder company names

Constraints:

- TypeScript + React + Tailwind CSS
- Responsive: stacks on mobile, side-by-side on desktop
- One file, one default export
